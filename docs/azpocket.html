AZPOCKET のファームウェア 書込み

このページはAZPOCKETのファームウェアをXIAO ESP32C6に書込むためのページです。
下記の手順でドライバのインストールとファームウェアの書き込みを行ってください。
Chromeで開く必要があります。



１．ドライバのインストール

下記サイトから「ESP-IDF Tools Installer」をダウンロードしてドライバのみをインストールして下さい。
※ 既にESP32S3を使用した事があるPCであればインストール不要です。

ESP-IDF Tools Installer



２．ファームウェアの書き込み

XIAO ESP32C6をPCに接続してBOOTボタンを押しながらRESETボタンを押して下さい。ESP32C6が書き込みモードで起動します。



書き込みモードになったら下記の書込みボタンをクリックしてファームウェアを書き込んでください。
「Write Complated.」と表示されたら書き込み完了です。Bluetoothからデバイスを検索するとAZPOCKETとして認識されます。
